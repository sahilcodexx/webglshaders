# webglshaders

> The shadcn/ui of shaders.

Browse, customize, and copy production-ready GLSL shaders.

## Shaders

- **Aurora** — flowing northern lights effect
- **Water** — realistic water surface with waves and foam
- **Noise** — animated fractal noise with domain warping
- **Galaxy** — spiral galaxy with stars and dust lanes
- **Gradient** — animated multi-stop color gradient

## Getting Started

```bash
pnpm dev
```
